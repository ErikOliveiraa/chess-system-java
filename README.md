# ♟️ Chess System Java

> Sistema de xadrez desenvolvido em Java com foco na aplicação de Programação Orientada a Objetos (POO), modelagem de domínio e implementação de regras de negócio complexas.

Este projeto foi desenvolvido como parte dos estudos em Java para consolidar conhecimentos em desenvolvimento orientado a objetos. A aplicação simula uma partida completa de xadrez no terminal, implementando as principais regras do jogo através de uma arquitetura organizada e código modular.

# 🏛 Arquitetura

O projeto foi estruturado seguindo princípios de separação de responsabilidades.

```
src
│
├── application
├── boardgame
├── chess
│   ├── pieces
│   ├── ChessMatch
│   ├── ChessPiece
│   ├── ChessPosition
│   └── Color
```

Cada pacote possui uma responsabilidade específica, facilitando manutenção, reutilização e evolução do sistema.
